
# Employee Shift Management System

A full-stack MERN application for managing employees, shift scheduling, attendance tracking, and leave requests with role-based access control.

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

---

## 👥 Roles & Access

### Admin
- Full access
- Manage employees
- Manage shifts
- View attendance
- Approve / Reject leaves

### Manager
- Manage employees
- Manage shifts
- View attendance
- Approve / Reject leaves

### Employee
- View assigned shifts
- Mark attendance
- Apply leave
- View own records

---

## ✨ Features

### Authentication
- Register
- Login
- JWT Token Based Authentication
- Role Based Access Control

### Employee Management
- Add Employee
- View Employees
- Delete Employee

### Shift Management
- Create Shift
- View Shift List

### Attendance
- Check In
- Check Out
- Auto Working Hours Calculation
- Attendance History

### Leave Management
- Apply Leave
- Approve Leave
- Reject Leave
- View Leave Requests
